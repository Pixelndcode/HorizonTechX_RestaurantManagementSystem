Restaurant Management System

Overview

This project is a backend Restaurant Management System developed using Flask and SQLite. It helps manage restaurant operations such as menu items, customer orders, and table reservations.

Features

- Add and manage menu items
- Place customer orders
- Reserve restaurant tables
- Store data using SQLite database
- Backend APIs built with Flask

Technologies Used

- Python
- Flask
- SQLite

Project Structure

- app.py
- restaurant.db
- README.md
- requirements.txt

How to Run

1. Install Flask
2. Run the app.py file
3. Start the Flask server
4. Access the available API routes

Learning Outcomes

- Backend development with Flask
- Database integration using SQLite
- API route creation
- Order and reservation management
